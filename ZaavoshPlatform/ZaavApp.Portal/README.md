# ZaavApp.Portal

Portal application for Zaavosh Platform built with Nuxt (Vue 3).

## Technology Stack
- **Framework**: Nuxt.js (Vue 3)
- **Styling**: Bootstrap 5.3 RTL
- **API Communication**: Axios
- **State Management**: Pinia

## Features
- Multi-language support (fa-IR, en-US)
- Responsive design
- Integration with ZaavApp.Api
- JWT authentication
- Persian calendar support

## Project Structure
```
ZaavApp.Portal/
├── pages/
├── components/
├── services/
├── layouts/
├── assets/
├── public/
└── nuxt.config.ts
```

## Setup Instructions
```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
